<h1 align="center">📚 z-reader-mini</h1>

> 基于 [z-reader](https://github.com/aooiuu/z-reader) 缩减版，只支持本地epub（自用）


---

## 🎉 功能

- 翻页逻辑：`j`上一页 `k`下一页，并自动翻章
- 小说阅读章节进度保存（翻页触发，直接点击章节不保存）
- 本地book文件夹不存在，自动建立，防止打不开资源管理器
